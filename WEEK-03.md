## O que aprendi
- plt.bar() gera barras verticais
- plt.barh() gera barras horizontais (melhor para nomes longos)
- plt.savefig() salva o gráfico como PNG
- JOIN cruza duas tabelas usando uma chave em comum
- ON define qual coluna liga as duas tabelas
- SUM(amount) soma valores agrupados por categoria
- HTML + CSS básico para montar um dashboard visual
- grid-template-columns divide a tela em colunas no CSS
